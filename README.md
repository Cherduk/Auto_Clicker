# Auto_Clicker

Программа для автоматического поиска изображения на экране и клика по найденной области.

## Описание

`Auto_Clicker` — это Windows-приложение с графическим интерфейсом на `tkinter`, которое:

- загружает шаблоны изображений из указанной папки
- ищет совпадения на экране с помощью OpenCV
- автоматически кликает по найденному элементу
- при необходимости выполняет прокрутку колесом мыши
- повторяет поиск через заданные интервалы времени

Программа подходит для автоматизации повторяющихся действий, где элемент нужно находить не по координатам, а по его внешнему виду.

## Возможности

- поиск элемента на экране по шаблону изображения
- поддержка нескольких шаблонов в одной папке
- настройка точности поиска (`Confidence`)
- настройка минимальной и максимальной задержки между кликами
- поиск в цвете или в режиме `Grayscale`
- поиск на всех мониторах
- прокрутка колесом мыши после клика
- простой графический интерфейс

## Интерфейс

В программе доступны следующие параметры:

- **Confidence** — уровень совпадения изображения от `0.01` до `1.0`
- **Min sleep seconds** — минимальная задержка между действиями
- **Max sleep seconds** — максимальная задержка между действиями
- **Templates directory** — папка с шаблонами изображений
- **Grayscale** — поиск в оттенках серого
- **Scroll amount in clicks** — величина прокрутки после клика  
  `Negative = up, positive = down`
- **Scan all monitors** — поиск на всех подключённых мониторах
